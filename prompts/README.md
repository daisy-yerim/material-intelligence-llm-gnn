# Prompt templates

This folder contains reusable prompt templates for the public Material Intelligence workflow.

- `cluster_interpretation_prompt.md`: summarize semantic clusters into business issue labels.
- `recommendation_explanation_prompt.md`: explain why a recommended entity or issue is related to an input message.
- `rag_relation_recommendation_template.md`: template for retrieval-augmented relation recommendation.

These prompts are intentionally separated from notebooks so that they can be reused, reviewed, and versioned independently.
